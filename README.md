# Car Sales Dashboard

A clean, responsive dashboard built with HTML, CSS, JavaScript, and Bootstrap. It loads the provided CSV dataset and provides filtering, sorting, pagination, highlighting, and export features.

## Features
- Filters: search, vehicle type, fuel, transmission, seller, owner, year range
- KPIs: totals and averages
- Charts: fuel mix and selling price trend
- Table tools: column visibility, sorting, pagination, row/column highlight
- Export: download filtered data as CSV

## How to Run
1. Start a local server in this folder:
   `python -m http.server`
2. Open `http://localhost:8000` in your browser.
